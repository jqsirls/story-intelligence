# PDF Version - Instructions

This folder contains HTML versions of all PRIVO submission documents that can be easily converted to PDF.

## Quick PDF Generation (Recommended)

### Method 1: Browser Print to PDF

1. **Open HTML files:**
   ```bash
   open pdf-version/*.html
   ```

2. **For each HTML file:**
   - File → Print (Cmd+P)
   - Click "PDF" dropdown
   - Select "Save as PDF"
   - Save with `.pdf` extension (replace `.html` with `.pdf`)

### Method 2: Batch Convert with Browser

1. Open all HTML files in browser tabs
2. Use browser's print dialog
3. Save each as PDF with matching name
