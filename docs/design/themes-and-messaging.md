## Messaging Hierarchy

### Hierarchy Levels

**Level 1: Core Message (Always Lead With)**
- Award-caliber quality
- Personalization (Story Intelligence™)
- Therapeutic/developmental value

**Level 2: Supporting Messages (Reinforce Core)**
- Safety and compliance
- Multi-platform access
- Family bonding

**Level 3: Feature Messages (Details)**
- Specific features (voice, video, smart home)
- Technical capabilities
- Integration options

### Message Flow

**Ideal Flow:**
1. **Hook:** Award-caliber quality or personalization
2. **Value:** Benefits to child and family
3. **Differentiation:** Story Intelligence™ or therapeutic value
4. **Proof:** Examples, testimonials, or features
5. **Call to Action:** Try free tier or learn more

## Message Testing Guidelines

### Testing Criteria

**Clarity:**
- Is the message clear and understandable?
- Does it communicate value immediately?
- Is jargon avoided?

**Relevance:**
- Does it address audience needs?
- Is it relevant to the use case?
- Does it resonate with the target audience?

**Differentiation:**
- Does it differentiate from competitors?
- Is the unique value clear?
- Does it position Storytailor uniquely?

**Authenticity:**
- Does it feel genuine and authentic?
- Is it consistent with brand voice?
- Does it avoid salesy language?

### Testing Process

1. **Draft message** (following framework)
2. **Review against criteria** (clarity, relevance, differentiation, authenticity)
3. **Test with target audience** (if possible)
4. **Refine based on feedback**
5. **Finalize and use consistently**
