# Storytailor Embeddable Widget - Marketing Information

**Last Updated**: 2025-12-17

## Value Proposition

The Storytailor Embeddable Widget enables partners to integrate Storytailor into their websites in minutes with zero-code setup, providing a beautiful, responsive chat/voice interface with customizable theming and privacy controls.

## Key Marketing Messages

- **"Integrate in Minutes"**: Add Storytailor to your website in minutes with just a script tag. No custom development required - widget handles all UI and functionality.
- **"Zero-Code Setup"**: Embed Storytailor with zero code. Just add a script tag and configure - that's it. Perfect for non-technical partners.
- **"Consistent Experience"**: Provide consistent Storytailor experience across all partner websites. Familiar interface, reliable performance, automatic updates.

## Target Audience Messaging

### For Partners

- "Add Storytailor to your website in minutes. Just embed a script tag, configure your theme, and you're done. No custom development required."
- "Customize Storytailor to match your brand. Custom logo, colors, and name - all while maintaining the Storytailor experience your users expect."
- "Privacy-first integration. COPPA mode, parental controls, and configurable data retention - all built-in for compliance."

### For Web Developers

- "Quick integration for testing and prototyping. Embed Storytailor widget in minutes, test functionality, then decide on full integration."
- "Easy customization with theming system. Predefined themes or custom branding - choose what works for your project."
- "Production-ready widget with comprehensive features. Error handling, event system, and privacy controls - all built-in."

### For Website Platform Users

- "Works on your favorite platform. Webflow, Framer, Wix, or custom websites - Storytailor widget works everywhere."
- "Consistent experience across platforms. Same Storytailor interface and functionality, regardless of website platform."
- "Automatic updates. Widget is maintained by Storytailor - you get bug fixes and new features automatically."

## Unique Selling Propositions (USPs)

1. **Zero-Code Integration**: Script tag integration enables non-technical partners to integrate Storytailor
2. **Quick Setup**: Integration in minutes, not days or weeks
3. **Theming System**: Predefined themes and custom branding for visual customization
4. **Multi-Platform Support**: Works on Webflow, Framer, Wix, and custom websites
5. **Production-Ready**: Production-tested widget with comprehensive features and error handling

## Use Cases

### Use Case 1: Partner Website Integration

**Scenario**: Partner wants to add Storytailor to their Webflow website

**How Widget Helps**:
- Partner adds script tag to Webflow custom code block
- Configures theme and branding
- Widget is live in minutes
- No custom development required

**Value**: Quick integration, zero maintenance, consistent experience

### Use Case 2: Quick Testing

**Scenario**: Developer wants to quickly test Storytailor integration

**How Widget Helps**:
- Developer embeds widget in test page
- Tests functionality immediately
- No need to build custom UI
- Fast iteration and testing

**Value**: Rapid prototyping, quick testing, fast iteration

### Use Case 3: Consistent Brand Experience

**Scenario**: Partner wants Storytailor with custom branding

**How Widget Helps**:
- Partner configures custom branding (logo, colors, name)
- Widget applies branding while maintaining Storytailor functionality
- Consistent brand experience across partner website

**Value**: Brand consistency, customization, professional appearance
