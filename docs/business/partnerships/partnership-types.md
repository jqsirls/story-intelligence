# Partnership Types

**Last Updated**: 2025-12-14  
**Audience**: Business Development | Partnerships | Enterprise Sales  
**Status**: ✅ Production Ready

## Overview

This document describes the various types of partnerships available in the Storytailor Partnership Program, including technology partnerships, solution partnerships, strategic partnerships, OEM partnerships, and channel partnerships.

## Technology Partnerships

### App Developers

**Description:**
Developers building mobile or web applications who want to integrate Story Intelligence™ storytelling capabilities.

**Use Cases:**
- Children's apps adding storytelling features
- Educational apps integrating story creation
- Entertainment apps with narrative elements
- Family apps with interactive content

**Integration Methods:**
- REST API integration
- Mobile SDKs (iOS, Android, React Native)
- Web SDK integration
- Webhook event system

**Benefits:**
- Access to Story Intelligence™ technology
- Technical integration support
- Co-marketing opportunities
- Revenue share opportunities

**Code References:**
- `docs/integration-guides/mobile-sdk.md` - Mobile SDK integration
- `docs/integration-guides/web-sdk.md` - Web SDK integration
- `docs/storytailor/partner_integration.md` - Complete integration guide
