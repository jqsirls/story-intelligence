## Overview

Security and compliance framework for Storytailor. Provides privacy audit services, data subject rights, PII detection, and compliance verification.

## Features

- **Privacy Audit Service**: Comprehensive privacy compliance auditing
- **Data Subject Rights**: GDPR Article 15 (data export) and Article 17 (right to erasure)
- **PII Detection**: Automated PII detection and redaction
- **Compliance Verification**: COPPA, GDPR, and child safety compliance checks

## Installation

```bash
npm install @alexa-multi-agent/security-framework
```
