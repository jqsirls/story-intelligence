## Overview

Comprehensive health monitoring orchestrator for all agents and services in the Storytailor platform. Provides real-time health monitoring, alerting, and performance tracking.

## Features

- **Comprehensive Health Checks**: Lambda functions, external APIs, multi-agent system, database
- **Real-Time Monitoring Dashboard**: System overview, service details, performance metrics
- **Intelligent Alerting**: Threshold-based alerts with severity classification
- **Performance Analytics**: Health scores, uptime tracking, response time analytics

## Installation

```bash
npm install @storytailor/health-monitoring
```
