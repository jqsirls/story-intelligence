## Overview

Token management service for authentication and authorization. Provides JWT token generation, validation, and refresh capabilities.

## Features

- **JWT Token Generation**: Access and refresh token generation
- **Token Validation**: Token verification and expiration checking
- **Token Refresh**: Secure token refresh mechanism
- **COPPA-Aware**: Age-based token permissions

## Installation

```bash
npm install @alexa-multi-agent/token-service
```
