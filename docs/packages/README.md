# Package Documentation

**Last Updated**: 2025-12-13  
**Audience**: Developers | Package Maintainers

## Overview

This directory contains documentation for all packages in the Storytailor monorepo, organized by category: supporting packages, service packages, and SDK packages.

## Package Categories

### Supporting Packages
Infrastructure and shared utilities used across the system:
- **[shared-types](./supporting/shared-types.md)** - TypeScript type definitions and gRPC schemas
- **[api-contract](./supporting/api-contract.md)** - API specifications and contracts
- **[ui-tokens](./supporting/ui-tokens.md)** - Design system tokens
- **[testing](./supporting/testing.md)** - Testing utilities and helpers
- **[monitoring](./supporting/monitoring.md)** - Monitoring and observability utilities

### Service Packages
Reusable services and frameworks:
- **[content-safety](./services/content-safety.md)** - Content safety and moderation
- **[event-system](./services/event-system.md)** - Event publishing and subscription
- **[health-monitoring](./services/health-monitoring.md)** - Health monitoring service
- **[security-framework](./services/security-framework.md)** - Security and compliance framework
- **[token-service](./services/token-service.md)** - Token management service

### SDK Packages
Platform-specific SDKs:
- **[web-sdk](./sdks/web-sdk.md)** - Web SDK (JavaScript/TypeScript)
- **[mobile-sdk-ios](./sdks/mobile-sdk-ios.md)** - iOS SDK (Swift)
- **[mobile-sdk-android](./sdks/mobile-sdk-android.md)** - Android SDK (Kotlin)
- **[mobile-sdk-react-native](./sdks/mobile-sdk-react-native.md)** - React Native SDK
- **[storytailor-embed](./sdks/storytailor-embed.md)** - Embeddable widget
