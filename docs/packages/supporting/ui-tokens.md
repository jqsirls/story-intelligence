# @storytailor/ui-tokens

**Package**: `@storytailor/ui-tokens`  
**Location**: `packages/ui-tokens/`  
**Version**: 1.0.0

## Overview

Design tokens and theme system for Storytailor platform consistency. Provides comprehensive design system ensuring visual consistency across all user interfaces.

## Features

- **Design Tokens**: Complete design system (colors, typography, spacing, etc.)
- **Multiple Formats**: TypeScript, CSS, SCSS, JSON exports
- **Brand Compliance**: Story Intelligence™ branding guidelines
- **Responsive**: Mobile-first breakpoints

## Installation

```bash
npm install @storytailor/ui-tokens
```

## Usage

See [UI Tokens README](../../../packages/ui-tokens/README.md) for complete usage examples.

## Key Features

- **Brand Colors**: Primary (#6366F1), Secondary (#EC4899), Accent (#F59E0B)
- **Typography**: Inter/Poppins font system
- **Spacing**: Consistent 0-24 scale (4px base unit)
- **Story Intelligence™ Branding**: Gradient system and messaging
