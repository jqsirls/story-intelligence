# @storytailor/api-contract

**Package**: `@storytailor/api-contract`  
**Location**: `packages/api-contract/`  
**Version**: 1.0.0

## Overview

OpenAPI specifications and type definitions for Storytailor platform APIs. Provides comprehensive API contracts ensuring type safety and consistent integration across all client applications.

## Features

- **OpenAPI Specifications**: Complete API documentation
- **TypeScript Types**: Generated from OpenAPI specs
- **Zod Schemas**: Runtime validation
- **API Configuration**: Base URLs, endpoints, error codes
